Run PARP1_Sep_2023_ensemble.ipynb to access validation data and the process of making and applying model enseble

Folder structure

bin - support functions
MD - molecular dynamics inputs and protein structures
run_PARP - scripts and config for running PARP screening with SMINA and GNINA
outputs - data including screening results for gnina and MM, validation results and compounds used for validation in .smiles and .mol format

The compleate dataset of outputs from the screening campaign can be provided on request (/outputs/PARP_sep_2023/gnina/sminaout/). Because of the size of the output files only a small sample is provided in this repository.
